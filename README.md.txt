# Avrupa Sosyo-Ekonomik Suç Analizi

Bu proje, Avrupa ülkelerinin sosyo-ekonomik göstergeleri ile şiddet suçları arasındaki ilişkiyi K-means kümeleme analizi ile inceler.

## Ekip

- Yaren Timur (230326887)
- Furkan Kemerpınar (230326817)

## Kullanılan Değişkenler
- GSYH (GDP)
- İşsizlik Oranı
- Askeri Harcamalar
- Kasıtlı Cinayet Oranları
